# react-native-magic-modal

a modal that can be used imperatively from anywhere!

## Installation

```sh
npm install react-native-magic-modal
```

## Usage

```js
import { multiply } from "react-native-magic-modal";

// ...

const result = await multiply(3, 7);
```
